# Naila-Projects
🔢 Binary Calculator (8086 Assembly Language)

📌 Overview
This project is a **Binary Calculator** developed using **8086 Assembly Language** as part of the **Computer Organization & Assembly Language (COAL)** course.

It performs both **arithmetic and logical operations** on binary numbers (up to 4 bits) using a menu-driven interface.

---

🎯 Objectives
- Understand low-level programming
- Perform binary arithmetic operations
- Implement logical/bitwise operations
- Work with interrupts (INT 21h, INT 16h)

---

⚙️ Technologies Used
- Assembly Language (8086)
- MASM / TASM
- DOSBox
- Windows OS

---

🚀 Features

 🔹 Arithmetic Operations
- Addition
- Subtraction
- Multiplication
- Division
- Exponent
- Factorial

 🔹 Logical Operations
- AND
- OR
- NOT
- 1’s Complement
- 2’s Complement

---

 🧠 Key Concepts
- Register manipulation
- Interrupt handling
- Binary to decimal conversion
- Procedure-based modular programming

---


 🏁 Conclusion
This project helped in building a strong understanding of:
- Assembly Language
- Binary operations
- Low-level system programming
